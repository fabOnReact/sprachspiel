# Environment

```
Ruby 2.5.0
Rails 5.1 
```

# Installation instructions

```
git clone git@github.com:fabriziobertoglio1987/sprachspiel.git
cd sprachspiel
rbenv local 2.5.0
bundle install
rails s
rails db:setup
```
you find the app at `localhost:3000`
